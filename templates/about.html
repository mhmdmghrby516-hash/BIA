About the upgraded project

This project now behaves like a real recommender-system demo rather than a storefront with precomputed JSON results. The application loads the raw dataset, cleans ratings and behavior, builds a hybrid baseline, and then applies a genetic algorithm to improve the final top-5 list.

System pipeline

  • Clean and aggregate users, products, ratings, and behavior.
  • Build user profiles from categories, price preference, and interactions.
  • Score products with content, price, collaborative, popularity, and rating signals.
  • Run a genetic algorithm over the candidate pool to improve diversity and list quality.
  • Evaluate offline with holdout metrics saved in docs/evaluation_summary.json.
